Legacy Pack Revival Downloader

How to Install/Update:
1. Download the core pack, and each of the packs you want using the buttons below.
2. Extract all of the 7z files to the root of your SD card (use 7-Zip or WinRAR!). Always say yes if asked to replace files.
3. If the setup is complete, you can now play Legacy Pack Revival through Riivolution. Make sure to enable online play!
When updating, just download the latest version of each pack. Not every latest version will have the same version number. This is okay.
